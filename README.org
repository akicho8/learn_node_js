* Node.js の練習用

  [[https://raw.github.com/akicho8/learn_node_js/master/screenshot.png]]

** 実行方法

: node app
: open http://localhost:3000/

** 最初にやったこと(よくわかってない)

: cd ~/src
: npm install express ejs -g
: express -e learn_node_js
: cd learn_node_js
: npm install
: npm link express

** バージョン

: $ node -v
: v0.9.1-pre
: $ npm -v
: 1.1.49
: $ express --version
: 3.2.6
